### ‚öôÔłŹ How It Works

**LDR #1 Path ‚ÄĒ Relay + Buzzer + RGB channel 1:**
```
Light ‚ÜĎ ‚Üí LDR#1 resistance ‚Üď ‚Üí Voltage divider midpoint ‚ÜĎ
‚Üí 2N2222 base voltage ‚ÜĎ ‚Üí Transistor ON
‚Üí Relay coil energized (1N4148 protects)
‚Üí Relay NO contact closes
‚Üí Buzzer sounds + RGB channel 1 ON
```

**LDR #2 Path ‚ÄĒ RGB channel 2 only:**
```
Light ‚ÜĎ ‚Üí LDR#2 resistance ‚Üď ‚Üí Voltage divider midpoint ‚ÜĎ
‚Üí RGB channel 2 pin voltage ‚ÜĎ ‚Üí RGB channel 2 ON
```

> Potentiometers allow independent threshold adjustment for each sensor.
